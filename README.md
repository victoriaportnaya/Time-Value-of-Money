### Internal Return Rate (IRR)

–¢–µ–ø–µ—Ä –Ω–µ—Ö–∞–π —É –Ω–∞—Å —î —Ç–∞–∫–∞ —ñ–Ω—Ñ–æ—Ä–º–∞—Ü—ñ—è: –ø–æ—á–∞—Ç–∫–æ–≤–∞ —ñ–Ω–≤–µ—Å—Ç–∏—Ü—ñ—è (—è–∫—É –º–∏ –∑–∞–ø–∏—Å—É—î–º–æ –∑—ñ –∑–Ω–∞–∫–æ–º "-"), —ñ –≥—Ä–æ—à–æ–≤–∏–π –ø–æ—Ç—ñ–∫ —É —Ñ–æ—Ä–º–∞—Ç—ñ. –ó–∞–ø–∏—Å—É—î–º–æ —ó—Ö –≤ list [ ]. –ù–∞—à–µ –∑–∞–≤–¥–∞–Ω–Ω—è - —Ä–æ–∑—Ä–∞—Ö—É–≤–∞—Ç–∏ **IRR**.

### –ü—Ä–∏–≤–µ–¥–µ–Ω–Ω—è –≤–∞—Ä—Ç–æ—Å—Ç—ñ: Present Value & Future Value

#### Present Value

–î–ª—è —Ä–æ–∑—Ä–∞—Ö—É–Ω–∫—É PV (present value) –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î–º–æ —Ñ—É–Ω–∫—Ü—ñ—é **npv.pv()**. –ù–∞ –≤—Ö—ñ–¥ –≤–æ–Ω–∞ –ø—Ä–∏–π–º–∞—î **–æ–±–æ–≤'—è–∑–∫–æ–≤—ñ** –∞—Ä–≥—É–º–µ–Ω—Ç–∏:
1. **rate** - –ø—Ä–æ—Ü–µ–Ω—Ç–Ω–∞ —Å—Ç–∞–≤–∫–∞ –Ω–∞ **–ø–µ—Ä—ñ–æ–¥** (array / scalar)
2. **nper** - –∫—ñ–ª—å–∫—ñ—Å—Ç—å –ø–µ—Ä—ñ–æ–¥—ñ–≤ (array / scalar)
3. **pmt** - —Ä–µ–≥—É–ª—è—Ä–Ω–∏–π –ø–ª–∞—Ç—ñ–∂ (array / scalar)
–Ü *–æ–ø—Ü—ñ–æ–Ω–∞–ª—å–Ω—ñ* –∞—Ä–≥—É–º–µ–Ω—Ç–∏: 
4*) *fv* - future value (array / scalar)
5*) *when* - –Ω–∞ —è–∫–∏–π –º–æ–º–µ–Ω—Ç –ø–ª–∞—Ç—ñ–∂ (–ø–æ—á–∞—Ç–æ–∫ –ø–µ—Ä—ñ–æ–¥—É - 1, –∫—ñ–Ω–µ—Ü—å - 0).

<br> **–ó–∞—É–≤–∞–∂–µ–Ω–Ω—è!** –Ø–∫—â–æ –¥–∞–ª—ñ –ø–ª–∞—Ç–µ–∂—ñ –Ω–µ –∑–¥—ñ–π—Å–Ω—é—é—Ç—å—Å—è —ñ –∑–Ω–∞—î–º–æ —Ç—ñ–ª—å–∫–∏ FV, —Ç–æ –∑–∞–ø–∏—Å—É—î–º–æ pmt = 0 —ñ –¥–∞–ª—ñ –≤–Ω–æ—Å–∏–º–æ –∞—Ä–≥—É–º–µ–Ω—Ç fv.

**–†–æ–∑–≥–ª—è–Ω–µ–º–æ –ø—Ä–∏–∫–ª–∞–¥**
<br>
–Ø–∫–∞ PV (–Ω–∞–ø—Ä–∏–∫–ª–∞–¥, –ø–æ—á–∞—Ç–∫–æ–≤–∞ —ñ–Ω–≤–µ—Å—Ç–∏—Ü—ñ—è) —ñ–Ω–≤–µ—Å—Ç–∏—Ü—ñ—ó, —è–∫–∞ –ø–æ–≤–∏–Ω–Ω–∞ —Å–∫–ª–∞—Å—Ç–∏ 350000 –≥—Ä–Ω. –ø—ñ—Å–ª—è 10 —Ä–æ–∫—ñ–≤ –∑–∞–æ—â–∞–¥–∂–µ–Ω—å –ø–æ 2000 –≥—Ä–Ω —â–æ–º—ñ—Å—è—Ü—è?
<br> –ü—Ä–∏–ø—É—Å—Ç–∏–º–æ, —â–æ –≤—ñ–¥—Å–æ—Ç–∫–æ–≤–∞ —Å—Ç–∞–≤–∫–∞ —Å—Ç–∞–Ω–æ–≤–∏—Ç—å 5% (—Ä—ñ—á–Ω–∏—Ö) –∑ —â–æ–º—ñ—Å—è—á–Ω–∏–º –Ω–∞—Ä–∞—Ö—É–≤–∞–Ω–Ω—è–º.
<br> **–ü–µ—Ä–µ–ø–∏—à–µ–º–æ –¥–∞–Ω—ñ —É –≤–∏–≥–ª—è–¥—ñ –∞—Ä–≥—É–º–µ–Ω—Ç—ñ–≤ —ñ —Å–∫–æ—Ä–∏—Å—Ç–∞—î–º–æ—Å—è —Ñ—É–Ω–∫—Ü—ñ—î—é.**


```python
rate = 0.05 # –Ω–µ –∑–∞–±—É–≤–∞—î–º–æ –ø—Ä–æ —Ñ–æ—Ä–º–∞—Ç float! —Ä–æ–∑–¥—ñ–ª–∏–º–æ –¥–∞–ª—ñ –Ω–∞ 12, —â–æ–± –æ—Ç—Ä–∏–º–∞—Ç–∏ rate per period
```

#### Future Value

–ú–∞–π–∂–µ –∞–Ω–∞–ª–æ–≥—ñ—á–Ω–æ, –¥–ª—è —Ä–æ–∑—Ä–∞—Ö—É–Ω–∫—É FV (future value) –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î–º–æ —Ñ—É–Ω–∫—Ü—ñ—é **npv.fv()**. –ù–∞ –≤—Ö—ñ–¥ –≤–æ–Ω–∞ –ø—Ä–∏–π–º–∞—î **–æ–±–æ–≤'—è–∑–∫–æ–≤—ñ** –∞—Ä–≥—É–º–µ–Ω—Ç–∏:

3. **pmt** - —Ä–µ–≥—É–ª—è—Ä–Ω–∏–π –ø–ª–∞—Ç—ñ–∂ (array / scalar)

<br> **–ü–µ—Ä–µ–ø–∏—à–µ–º–æ –¥–∞–Ω—ñ —É –≤–∏–≥–ª—è–¥—ñ –∞—Ä–≥—É–º–µ–Ω—Ç—ñ–≤ —ñ —Å–∫–æ—Ä–∏—Å—Ç–∞—î–º–æ—Å—è —Ñ—É–Ω–∫—Ü—ñ—î—é.**

### –ß–∏—Å—Ç–∞ –ø–æ—Ç–æ—á–Ω–∞ –≤–∞—Ä—Ç—ñ—Å—Ç—å: Net Present Value

–¢—É—Ç —É—Å–µ –¥—É–∂–µ –ø—Ä–æ—Å—Ç–æ. –Ø–∫ –Ω–µ—Å–∫–ª–∞–¥–Ω–æ –∑–¥–æ–≥–∞–¥–∞—Ç–∏—Å—è, –º–∏ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î–º–æ —Ñ—É–Ω–∫—Ü—ñ—é **npf.npv()**. –Ñ —Ç—ñ–ª—å–∫–∏ –¥–≤–∞ –∞—Ä–≥—É–º–µ–Ω—Ç–∏:

**–†–æ–∑–≥–ª—è–Ω–µ–º–æ –ø—Ä–∏–∫–ª–∞–¥**
<br> –ü—Ä–æ–∞–Ω–∞–ª—ñ–∑—É—î–º–æ –ø–æ—Ç–µ–Ω—Ü—ñ–π–Ω–∏–π –ø—Ä–æ–µ–∫—Ç –∑ –ø–æ—á–∞—Ç–∫–æ–≤–æ—é —ñ–Ω–≤–µ—Å—Ç–∏—Ü—ñ—î—é 100 000 –≥—Ä–Ω. —ñ –ø—Ä–æ–≥–Ω–æ–∑–æ–≤–∞–Ω–∏–º–∏ –≥—Ä–æ—à–æ–≤–∏–º–∏ –ø–æ—Ç–æ–∫–∞–º–∏ 50 000, 8 000, 15000 —ñ 60 000 –≤ –∫—ñ–Ω—Ü—ñ –∫–æ–∂–Ω–æ–≥–æ –ø–µ—Ä—ñ–æ–¥—É, –¥–∏—Å–∫–æ–Ω—Ç–æ–≤–∞–Ω–∏–º–∏ –∑–∞ —Å—Ç–∞–≤–∫–æ—é 7% –∑–∞ –ø–µ—Ä—ñ–æ–¥. –ó–Ω–∞–π—Ç–∏ NPV.
<br> **–ü–µ—Ä–µ–ø–∏—à–µ–º–æ –¥–∞–Ω—ñ —É –≤–∏–≥–ª—è–¥—ñ –∞—Ä–≥—É–º–µ–Ω—Ç—ñ–≤ —ñ —Å–∫–æ—Ä–∏—Å—Ç–∞—î–º–æ—Å—è —Ñ—É–Ω–∫—Ü—ñ—î—é.**

### –ê–Ω—É—ó—Ç–µ—Ç–Ω–∏–π –ø–ª–∞—Ç—ñ–∂
